### Issue #110: Code review round 18 (PR #111 ‚Äî merged + deployed)

5. **Telegram/Discord link validation** (IMPORTANT) ‚Äî Server accepted bare `https://t.me/` with no path. Replaced `startsWith` with strict regex requiring path characters after domain.

6. **Reputation userId validation** (IMPORTANT) ‚Äî `userId=0` or negative passed `isNaN` guard, returning misleading zeroed stats. Added `< 1` check.

7. **fetchRatings re-triggered** (IMPORTANT) ‚Äî Client re-fetched ratings on every `setListing` call. Added `ratingsFetchedRef` guard to fire only once.
